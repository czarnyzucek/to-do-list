#  TO-DO LIST
Simple to-do list.

##  How to use
1.  Write the task you need to complete
2.  Click on "Dodaj zadanie" button
3.  Click on the green button if your task is completed
4.  Click on the red button if you want to delete your task

##  Used Technologies
- HTML
- CSS
- JavaScript 
- BEM convention
- Grid
- Flexbox
